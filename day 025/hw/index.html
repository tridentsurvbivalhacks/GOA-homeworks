What is html





Html is

  • HyperText
  • Markup
  • Language

HyperText Means:

Text with links (or references) — Hypertext is text that doesn’t just sit statically on a page. It can include hyperlinks, which connect one document to another or even to another location within the same document.


Markup means:

Special tags around text — You “mark up” plain text with tags like ( paragraph tag, heading tag, senematic elements, anchor tags and more... ) to tell the browser how to display or structure it.


Language means:

A system of rules and symbols — Just like spoken languages use grammar and vocabulary, HTML uses tags and syntax to describe structure.


A formal way to communicate with computers — HTML tells web browsers how to interpret and display content.



in short


  • HyperText → text with links
  • Markup → tags that structure the text
  • Language → the standardized system of rules for writing those tags
