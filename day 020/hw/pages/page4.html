Python basics

Python is a high-level interpreted programming language designed to be simple and readable. You write instructions line by line, and Python executes them directly without compiling first. Code blocks in Python are defined by indentation rather than braces or keywords. A comment starts with a hash symbol (#) and is ignored by the interpreter. You can create variables by assigning values to names. For example, x = 10 stores the number 10 in x. You do not need to declare a type because Python figures it out automatically. Data can be numbers, text strings in quotation marks, Boolean values True or False, or more complex structures such as dictionaries and sets. Conditional statements let you run code depending on a condition. You use if, elif, and else to control which block of code executes. Loops repeat code. A for loop runs through a sequence of numbers or items, while a while loop continues until a condition becomes false. Functions are reusable pieces of code you define with the def keyword. They can take inputs called parameters and can return outputs using the return statement. Python has many built-in functions and libraries. You can bring extra functionality into your program using the import statement, such as import math to work with mathematical functions. Errors can be managed with try and except blocks so your program does not crash when something unexpected happens. Finally, Python is also an object-oriented language, meaning you can define your own types called classes, bundle data and behavior together, and create instances (objects) from them.
